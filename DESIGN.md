# NanoAI Landing Page Design

## Brand Identity
- **Name**: NanoAI
- **Tagline**: The Universal Offline AI Runtime.
- **Colors**:
  - Primary: Deep Tech Blue (#0A192F)
  - Secondary: Electric Cyan (#64FFDA)
  - Accent: Matrix Green (#00FF41)
- **Typography**: Inter (Modern, Clean, Sans-serif)

## Layout Sections
1. **Hero**:
   - Headline: "AI Everywhere, Cloud Nowhere."
   - Sub-headline: "Run Llama, Gemma, and ONNX models locally with NPU acceleration on Android, Windows, and Embedded devices."
   - Primary CTA: "Get the SDK"
   - Secondary CTA: "View Roadmap"
2. **Hardware Compatibility Grid**:
   - Icons for Qualcomm Hexagon, NVIDIA CUDA, Apple Metal, MediaTek APU.
3. **SDK Snippets**:
   - Multi-tab code block showing Kotlin, Python, and Rust examples.
4. **Pricing Tiers**:
   - Community (Free), Starter (9), Business (99), Enterprise (Custom).
5. **Roadmap**:
   - Visual timeline from v1 to v5.

## UI Components
- **Navbar**: Logo, Docs, Pricing, GitHub Link.
- **Card**: Hover effects with glowing borders.
- **Code Block**: Dark theme with syntax highlighting.
