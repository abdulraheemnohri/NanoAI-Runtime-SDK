# NanoAI Runtime SDK Roadmap

## v1: Foundation
- Support for **ONNX**, **TFLite**, and **GGUF**.
- Support for **Android** and **Windows**.
- Initial **CPU** inference optimization.

## v2: GPU Acceleration
- Introduction of **GPU acceleration**.
- **Vulkan** backend support.
- **OpenCL** backend support for cross-vendor GPU usage.

## v3: NPU Optimization
- Native **NPU support** for specialized AI hardware (Qualcomm, MediaTek, Samsung).
- Edge deployment tools for optimized model conversion.

## v4: Advanced Connectivity
- **Distributed inference**: Split workloads across multiple local devices.
- **Device-to-device AI networking**.

## v5: Universal Platform
- **AI Runtime OS layer**.
- **Universal edge AI platform** for seamless deployment across all hardware.
